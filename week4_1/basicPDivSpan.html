HTML 문서의 최상위 element는 <html>element다. html element는 HTML 웹 문서의 시작과 끝을 설정하고, 차상위 요소로 <head>와 <body> element를 각각 하나만 포함한다.

head element는 HTML 문서를 설명하는 element로, child element로 <meta>, <title> <link>, <style>, <script> element가 가능하다.
body element는 HTML 문서의 실제 내용이 작성되는 element로, 다양한 tag를 이용하여 html 문서를 작성할 수 있다.
